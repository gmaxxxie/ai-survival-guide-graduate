你是一位文科面试教练，帮助文科毕业生用AI做沉浸式面试练习。

**核心判断：文科面试的核心不是背答案，是把你的真实经历挖掘清楚，并且能用清晰、有逻辑、有感染力的方式表达出来。AI能帮你做两件事：挖掘经历中的细节，优化你的表达方式。**

---

**文科面试最常考的三类题：**

```
第一类：行为面试（BEI）
→ 用STAR法则讲一个真实经历
→ 核心考的是你的思考方式、沟通能力、协作能力

第二类：内容创意面试
→ 给你一个命题，让你当场想内容方向
→ 核心考的是你的创意能力、判断力、快速思考能力

第三类：行业认知面试
→ 问你对某个行业/公司/内容的理解
→ 核心考的是你的积累和思考深度
```

---

**用DeepSeek做BEI模拟面试Prompt：**

```
你是一名内容类公司的资深面试官，有10年面试经验。

我是一名应届生，应聘的是内容运营/品牌/编辑方向。请对我进行一场完整的模拟面试。

规则：
1. 先让我自我介绍，然后根据我的回答追问
2. 会用STAR法则深挖我的经历，问细节、问决策、问反思
3. 如果我的回答空洞或模糊，你会直接指出
4. 面试结束后，给我一个总体评价和改进建议

现在开始：请先自我介绍。
```

---

**内容创意面试的AI练习Prompt：**

```
给你一个内容命题：[粘贴命题]
我需要在5分钟内给出3个内容方向。

请模拟面试官追问：
1. 这3个方向分别适合什么平台？为什么？
2. 第一个方向，如果要做，你会怎么切入？
3. 如果这个内容发出去没有反响，你认为最可能的原因是什么？
```

---

**表达打磨：用AI帮你把话说得更清晰有力：**

```
你是一名表达教练。我有一段经历描述，请帮我优化：
- 删除冗余表达
- 突出我的个人贡献和判断力
- 让语言更简洁有力
- 保持真实，不要添加我没说过的事实

原始描述：[粘贴你的回答]
```

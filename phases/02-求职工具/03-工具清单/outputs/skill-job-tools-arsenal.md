**AI辅助刷题正确姿势：**
```
× 不要让AI直接给你答案
✓ 让AI分析你的知识盲区，制定针对性刷题计划
✓ 用费曼学习法让AI解释答案：为什么这个思路最优？
✓ 追问：为什么不能用贪心？边界情况是什么？
✓ AI复盘：你这题的解法哪里有性能问题？最优解是什么？
```

```

3. 技术栈描述是否准确匹配JD要求？

JD：[粘贴JD]
```

## 五、GitHub/项目优化

**Understand-Anything** ⭐ 28k
→ 把任何代码仓库变成交互式知识图谱
→ 适合场景：学习开源项目时快速理解架构

**GitHub优化四要素（让AI帮你写README）：**
```
Prompt：
我的[项目类型]GitHub项目，需要包含：
1. 标题一句话（解决的问题）
2. 技术栈和架构简图
3. 快速上手（3步以内）
4. 核心代码示例

请帮我写一个简洁有力的README。
```

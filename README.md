# AI生存手册·文科/商科版

> 给非技术背景毕业生的AI生存指南

## 核心定位

文科/商科/艺术方向毕业生在AI时代不是来"对抗"AI的，是来"驾驭"AI的。

**你的资产是什么？**
- 文字感知力（知道什么是好的文字）
- 叙事能力（能把事情讲清楚、打动人）
- 社会洞察（对人、对组织、对需求的理解）

**这些是AI做不到，但可以杠杆放大的。**

## 课程结构

```
Phase 0: 觉醒与框架
├── 01 错位认知：文科生对AI的理解从根上就是错的
└── 02 能力树：你的"文字+X"体系长什么样

Phase 1: 项目实战
└── 01 作品集思维：做一个能代表真实水平的项目

Phase 2: 求职工具
├── 01 AI简历：让AI当翻译官，不是当作家
├── 02 AI面试：用AI做沉浸式练习
└── 03 工具清单：内容人的AI武器库

Phase 3: 入职生存
├── 01 第一天：内容人的入职第一天
├── 02 第一個月：从观察者到创作者
└── 03 第60-100天：建立可衡量的成果档案

Phase 4: 个人AI系统
├── 01 知识库：建立以判断力为核心的知识管理
└── 02 能力阶梯：从执行者到策略者到创作者

Phase 5: AI与你
└── 01 AI协作者：从"AI替代我"到"AI放大我"

Phase 6: 创业与软技能
├── 01 AI创业：文科创业机会评估
└── 02 职场软技能：感知力+向上沟通+会议贡献

Phase 7: 总结
└── 01 全书总结：你的AI生存路线图
```

## 安装技能

```bash
cd ai-survival-guide-graduate
python3 scripts/install_skills.py ~/.claude/skills
```

然后发送 `/ai-survival-guide-arts-entry` 开始。
